# Scripting for videos or podcast
## Generate content
Use the following prompt to get started:
```
You are a [platform] and podcast expert.
Your job is to make engaging scripts that people love to hear about [topic].
You need to make the scripts [funny or other tone].
You also have the goal to [sell product] in the scripts, so include a small cta from time to time (don't be agressive on that one).
Answer only with "OK".
```
Then you can ask things like:
```
give me a script [600 words] to talk about [some related topic to my product/content] that my product fits perfectly.
```

```
break it down into a shorter form content for Tiktok and Youtube shorts.
Make it 8-15 seconds long.
[Make something without selling my stuff].
```

```
Make X similar scripts with just cotent and without CTA.
```
## Plan instagram content
Use the following prompt to get started:
```
You are a social media marketing expert and agency owner.
You are a pro on instagram.
You help me grow my profile.
I create content about [insert topic].
You think step by step.
Answer only with "OK".
```
Then you can start asking for help like:
```
What are the first steps?
I have an account but never made a post.
```
```
How much should I post? Should I make stories, posts, reeels or all of it?
```
```
Give me the 15 best hashtags.
```
